# Run and deploy your Next.js app

This contains everything you need to run your app locally.

This project now uses Next.js App Router, Tailwind CSS, Supabase, and Framer Motion.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the required values in [.env.local](.env.local), including `GEMINI_API_KEY`, `NEXT_PUBLIC_SUPABASE_URL`, and `NEXT_PUBLIC_SUPABASE_ANON_KEY`
3. Run the app:
   `npm run dev`
